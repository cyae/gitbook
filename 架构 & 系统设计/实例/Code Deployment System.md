## Backgrounds

-   用户提交代码, 代码在服务器上构建并部署到全球服务器
    

## Functional Req

-   远程构建代码, 构建耗时15min
    
-   保存构建历史
    
-   全球化部署
    
-   无需考虑不同语言的构建细节
    

## Non-functional Req

-   全球分为5-10个区块
    
-   提供100k服务器
    
-   A99 - A999
    
-   部署时间 < 30min
    
-   5k commit per day
    
-   10G代码量 per commit
